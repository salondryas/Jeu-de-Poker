### Ce projet permet de :

✔ Créer des cartes (numéro + couleur)
✔ Construire une main de 5 cartes
✔ Trier les cartes en ordre décroissant
✔ Détecter les combinaisons suivantes :
**Paire
Double paire
Brelan
Full (Brelan + Paire)
Carré
Couleur
Suite
Quinte flush
Quinte flush royale**
✔ Comparer deux mains complètes et indiquer laquelle gagne
✔ Gérer les kickers en cas d’égalité partielle
✔ Générer une description textuelle de chaque main
✔ Afficher clairement le gagnant dans la démonstration console
### État final du projet : STABLE et FONCTIONNEL
Tous les tests importants sont réussis
### Lancer le programme principal
Depuis IntelliJ :

Ouvrir le projet

Aller dans JeuPoker.java

Faire Run

Le programme va :

Demander deux mains
Afficher les cartes triées
Dire quelle main gagne
Afficher la combinaison gagnante
